Sorting and Arrays with MemoryVM

To introduce a little more complexity and explore some of the cooler features of MVM, we are going to write a simple sort. Bubblesort! It may be slow, but it's easy to understand, and easy enough to implement.

Bubblesort

In bubblesort, each element is compared with its neighbor and swapped if necessary. Unfortunately, it can't always sort the whole array in one pass. It must keep running until all of the elements have slowly made their way to their correct positions. In a C-like language, a typical implementation looks something like this: 

int len = 4;
int array[len] = {1, 2, 3, 4};
bool swapped = true;

while (swapped) {
    swapped = false;
    for (int i = 1; i < len; i++) {
        if (array[i - 1] > array[i]) {
            swap(&array[i - 1], &array[i]);
            swapped = true;
        }
    }
}
