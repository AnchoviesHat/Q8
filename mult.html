Multiplication with Quine8

As a gentle introduction to the Quine8 (Q8) tool, we'll start by multiplying two numbers. Because the Q8 instruction set does not contain a multiply operation, we'll have to write our own. The algorithm for multiplication is pretty easy, you probably learned it in grade school.

6 * 4 = 24

That can be expanded in one of two ways:

6 + 6 + 6 + 6 = 24
6, 4 times is 24
4 + 4 + 4 + 4 + 4 + 4 = 24
4, 6 times is 24

In a C-like language, they look like this:

int x = 0;
for (int i = 4; i > 0; i--) {
    x += 6;
}
int x = 0;
for (int i = 6; i > 0; i--) {
    x += 4;
}

Finished Program

 
; INIT
LOAD A 80 ; load x
LOAD B 81 ; load y
CMP A B ; is x > y?
JG 32 ; goto swap
; 9 - SAVE_VAR
STORE A 96 ; save min(x, y) as i
STORE B 112 ; save max(x, y) as increment
JMP 36 ; goto

; 32 - SWAP
SWP A B ; (x, y) -> (y, x)
JMP 9 ; goto save_var

; 36 - LOOP
LOAD A 96 ; get i
ISZERO A ; is i == 0?
JZ 255 ; goto exit
DEC A ; i--
STORE A 96 ; save new i
JMP 67 ; goto add increment

; 67 - ADD INCREMENT
LOAD A 97 ; read in cumulative value
LOAD B 112 ; read in increment
ADD A B ; cumulative = cumulative + increment
JERR 255 ; if overflow, exit
STORE A 97 ; save new cumulative
JMP 36 ; goto loop
