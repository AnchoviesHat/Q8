Graphics Fun with Q8

To mix things up a little, this project is no math, just visuals. Here we are going to write a little code to bounce a "ball" from one side of the screen to the other. We're going to do this the simple way, with two loops. One loop to go across the screen right, and one to go back left again. A keen reader might spot a way to make this a single loop, probably reducing the code redundancy considerably. I encourage you to explore that avenue as a fun little bonus challenge.

C-like Program

while (true) {
    int position = 240;
    for (int i = 15; i > 0; i--) {
        //draw 0 @ array[position]
		position++;
        //draw 1 @ array[position]
    }
    for (int i = 15; i > 0; i--) {
        //draw 0 @ array[position]
        position--;
        //draw 1 @ array[position]
    }
}

Assembly Program

; 0  - INC-INIT
inc-init:
LOAD A 128 ; read in ball start position
LOAD B 129 ; read in bounce size
STORE A 144 ; make copy of ball start position
STORE B 145 ; make copy of bounce size as new i
JMP inc-loop

; 32 - INC-LOOP
inc-loop:
LOAD A 145 ; read in i
ISZERO A ; is i == 0?
JZ dec-init
DEC A ; i--
STORE A 145 ; write new i

; 40 - INC-DRAW BALL
inc-draw:
SET A 0 ; load ball clear
STOREI A 144 ; clear old position
LOAD A 144 ; read position
INC A ; position++
STORE A 144 ; write position to grid
SET A 1 ; load ball
STOREI A 144 ; write ball to new position
JMP inc-loop

; 64 - DEC-INIT
dec-init:
LOAD A 129 ; read in bounce size
STORE A 145 ; make copy of bounce size as new i
JMP dec-loop

; 80 - DEC-LOOP
dec-loop:
LOAD A 145 ; read in i
ISZERO A ; is i == 0?
JZ inc-init
DEC A ; i--
STORE A 145 ; write new i

; 88 - DEC-DRAW BALL
dec-draw:
SET A 0 ; load ball clear
STOREI A 144 ; clear old position
LOAD A 144 ; read position
DEC A ; position--
STORE A 144 ; write position to grid
SET A 1 ; load ball
STOREI A 144 ; write ball to new position
JMP dec-loop

; DATA
$128 240 15 ; ball_start_pos, bounce_len
